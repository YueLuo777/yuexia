# 伏笔库

## 使用说明
本目录用于管理小说中的伏笔，包括：
- `已埋伏笔.md` - 已埋设但尚未回收的伏笔
- `已收伏笔.md` - 已回收的伏笔归档
- `伏笔总表.md` - 所有伏笔的总览

---

# 已埋伏笔

## 格式说明
每个伏笔按以下格式记录：

```markdown
### [伏笔名称]
- **埋设章节**: 第X章
- **埋设内容**: [具体描述]
- **预计回收**: 第X卷/第X章
- **重要性**: ⭐⭐⭐ (1-3星)
- **章节引用**: "引用原文..."
```

---

## 待回收伏笔

# 已收伏笔

## 格式说明
```markdown
### [伏笔名称]
- **埋设章节**: 第X章
- **回收章节**: 第Y章
- **伏笔内容**: [内容]
- **回收方式**: [如何回收]
- **效果评估**: [好/一般/需改进]
```
