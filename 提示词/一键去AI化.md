# 一键去AI化工作流

## 使用方法
1. 指定要处理的章节文件
2. AI 自动检测并修改AI痕迹
3. 生成润色后的版本

## 执行步骤

### 步骤1: 读取章节内容
读取指定章节文件

### 步骤2: AI痕迹检测

#### 高频AI词汇（必须删除/替换）
| AI词汇 | 替换方案 |
|--------|----------|
| 穿透时光 | 删除 |
| 指节发白 | 手指攥紧/捏得生疼 |
| 眼眸/眸子 | 眼睛 |
| 微微上扬 | 笑了笑/嘴角一翘 |
| 心中激荡 | 心里一震/心头一紧 |
| 某种意义上 | 删除 |
| 仿佛/似乎（过多） | 控制在每章2处以内 |
| 不禁/不由得（过多） | 控制在每章2处以内 |
| 首先/其次/最后 | 删除 |
| 缓缓说道 | 说道/道 |
| 淡淡地说 | 说/随口道 |
| 微微点头 | 点头 |
| 轻轻摇头 | 摇头 |

#### 结构性AI痕迹
- ❌ 过长的心理描写（超过3句）
- ❌ 过长的环境描写（超过2句）
- ❌ 重复的修饰词堆砌
- ❌ 过于工整的句式结构
- ❌ 缺少语气词的对话

### 步骤3: 网文化处理

#### 增加元素
- ✅ 语气词：哎、嗯、切、啧、哼、呵
- ✅ 口语化表达：整个人都不好了、无语凝噎
- ✅ 生理反应：腿软、冷汗、嘴角抽搐、眼皮跳
- ✅ 短句化处理：拆分长句为2-3个短句
- ✅ 环境小动作：咳嗽声、脚步声、风声

#### 节奏调整
- 对话前后增加空行
- 每段控制在5行以内
- 重要对话单独成行

### 步骤4: 保存处理结果
- 覆盖原文件 或
- 保存为新文件（添加`_去AI化`后缀）

### 步骤5: 生成处理报告

```markdown
# 去AI化处理报告

## 文件信息
- **原文件**: [路径]
- **处理时间**: YYYY-MM-DD HH:MM

## 修改统计
- **删除AI词汇**: X处
- **替换AI词汇**: X处
- **增加语气词**: X处
- **拆分长句**: X处
- **增加生理反应**: X处

## 主要修改
1. [位置]: [原文] → [修改后]
2. ...
```

## 去AI化对照表

### 动作描写
| AI写法 | 网文写法 |
|--------|----------|
| 他微微皱眉 | 他眉头一皱 |
| 缓缓转身 | 转过身来 |
| 轻轻叹息 | 叹了口气 |
| 淡淡一笑 | 笑了笑 |
| 眼神微变 | 眼皮一跳 |

### 心理描写
| AI写法 | 网文写法 |
|--------|----------|
| 心中涌起一股复杂的情绪 | 心里五味杂陈 |
| 他意识到了某种可能性 | 他猛然醒悟 |
| 一种难以名状的感觉袭来 | 总觉得不对劲 |

### 对话描写
| AI写法 | 网文写法 |
|--------|----------|
| 他缓缓说道 | 他说 |
| 她淡淡地回应 | 她道 |
| 他沉声道 | 他道 / "..." 他声音沉沉 |
