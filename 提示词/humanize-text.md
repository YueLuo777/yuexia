# Humanize Text 工作流

## 用途
将AI生成的文章处理成更符合人类写作风格的版本，去除明显的AI痕迹。

## 执行步骤

### 步骤1: 读取原文
读取需要处理的文章内容

### 步骤2: AI痕迹识别
识别以下AI特征：

#### 词汇层面
- 过度使用"首先、其次、最后"
- 大量"不禁、不由得、似乎、仿佛"
- 工整的排比句
- 过于华丽的修饰语

#### 结构层面
- 过于规整的段落结构
- 机械的总分总模式
- 缺少口语化表达
- 没有个人化特征

### 步骤3: 人性化处理

#### 词汇替换
| 原词 | 替换 |
|------|------|
| 不禁 | 忍不住/没忍住 |
| 不由得 | 下意识 |
| 首先/其次/最后 | 删除或自然过渡 |
| 似乎/仿佛 | 减少使用 |

#### 结构调整
- 打破规整的段落结构
- 增加口语化短句
- 添加语气词（嗯、啊、呢、嘛）
- 增加不规则的思维跳跃

#### 增加人类特征
- 偶尔的重复强调
- 口语化表达
- 个人化的小癖好
- 不完美的句式

### 步骤4: 输出结果
保存处理后的版本

## 检查清单
- [ ] 删除了总结式表达
- [ ] 减少了华丽修饰
- [ ] 增加了口语化元素
- [ ] 打破了规整结构
- [ ] 保持了内容完整性
